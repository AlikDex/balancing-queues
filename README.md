### Балансер очередей.

Старт:
```shell
composer dump-autoload && php ./test.php
```
1)
Не стал заморачиваться с нормализацией веса и приведением его к целому числу

Как примерно делать: проверить есть ли флоат, если есть, взять макс вес, поделить на минимальный флоат.
Будет множитель общий.

Затем помножить все на этот множитель, чтобы все числа были целые.
Возможно надо каждый раз номализовывать вес всех очередей, при добавлении новой в стратегию.

Но лучше ввести соглашение, что вес всегда будет одним типом, лучше целым числом (проще).

2)
Не стал заморачиваться с обработкой ситуации когда очередь с идом уже существует(повторное добавление).
Можно игнориовать или кинуть исключение, по ситуации.
